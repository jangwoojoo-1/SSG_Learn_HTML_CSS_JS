    반응형 디자인 무엇인지?
    많은 사이트가 다양한 화면 맞춰 자동으로 요소를 배치하는 웹페이즈를 반응형 웹이라고 함.
    - 기기 화면에 맞춰 요소가 자동으로 배치된 웹 페이지를 제공하므로 긱화면마다 따로  코드를 개발 관리하지 않아도 된다.
    개발 비용응ㄹ 절약할 수 있으며 사용자 편의성을 높일 수 있다.

    css 박스 모델은 기본적으로 모든 HTML을 감싸는 상자이다.
    - 요소(cotnent)_ : 텍스트, 사진 등 보여줄 대상
    - 패딩(padding) : 요소 주변 영역을 감싼다. 해당 부분은 투명색이다. 
    - 테두리 (border) :  요소와 패딩을 감싸는 테두리앋. 색상 변경 가능
    -마진(margin) : 테두리 밖의 영역르 감싼다. 투명색이다.
